# Vaishnavi Portfolio Website

A personal portfolio website built with React and Vite for deployment on Netlify.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Netlify

- Push this project to GitHub
- In Netlify, choose **Add new site > Import an existing project**
- Select the repository
- Build command: `npm run build`
- Publish directory: `dist`
- Deploy site

## Customize

Update these files:
- `src/App.jsx` for content
- `src/styles.css` for design
- Add your resume as `public/resume.pdf`
- Replace placeholder email and LinkedIn URL
